# E-commerce_Sales_Analysis

### Project Overview
This Data Analysis Project Aims to provide insight into the sales performance of a Fintech Based in Lagos in 2nd And 3rd Quaters of the Year 2021. By analysing various aspect of the sales data, we seek to identify trends, make data_driven recommendations and gain a deeper understanding of the companys performance.

### Data Source
The Dataset used for this analysis is the "Main_Service_Provider.xlsx"
[Download here](https://github.com/user-attachments/files/17994467/Main_Service_Provider_.xlsx)

 ### Data Cleaning
-   In the preparation of dataset for analysis we performed the following task:
1. Removal of duplicates
2. Handling Missing values with Vlookup
3. Change Values to their appropriate format

### Exploratory Data Analysis
The data was explored to enter key questions such as
- What was the monthly sales trend
- What were the top performing categories
- what were the peak sales period
